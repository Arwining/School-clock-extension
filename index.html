Step-by-Step Guide to Create Class Schedule Timer Extension

This guide will walk you through creating a simple browser extension that shows your next class and counts down to it. Follow each step carefully.

Step 1: Prepare the Files

Create a new folder on your computer, for example, MyScheduleExtension. Inside this folder, create the following files with the specified contents.

1. manifest.json

{
  "manifest_version": 3,
  "name": "Class Schedule Timer",
  "version": "1.0",
  "description": "Shows time until next class",
  "action": {
    "default_popup": "popup.html"
  }
}

2. popup.html

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8" />
<title>Next Class Timer</title>
<style>
  body { font-family: Arial, sans-serif; width: 200px; padding: 10px; margin: 0; background-color: #fff; }
  h1 { font-size: 1.2em; margin: 0 0 10px 0; }
  #schedule { margin-bottom: 10px; }
  #time { font-size: 1.5em; font-weight: bold; }
</style>
</head>
<body>
<h1>Next Class</h1>
<div id="schedule"></div>
<div id="time"></div>
<script src="popup.js"></script>
</body>
</html>

3. popup.js

const schedule = [
  { name: "Math", time: "09:00" },
  { name: "English", time: "10:30" },
  { name: "Science", time: "13:00" },
  { name: "History", time: "14:30" },
  { name: "PE", time: "16:00" }
];

function getNextClass() {
  const now = new Date();
  const nowMinutes = now.getHours() * 60 + now.getMinutes();
  let next = null;
  let minDiff = Infinity;
  for (const c of schedule) {
    const [h, m] = c.time.split(":").map(Number);
    const classMinutes = h * 60 + m;
    const diff = classMinutes - nowMinutes;
    if (diff >= 0 && diff < minDiff) {
      minDiff = diff;
      next = c;
    }
  }
  return { class: next, minutes: minDiff };
}

function update() {
  const next = getNextClass();
  const scheduleDiv = document.getElementById("schedule");
  const timeDiv = document.getElementById("time");
  if (next.class) {
    scheduleDiv.textContent = \`Next: \${next.class.name} at \${next.class.time}\`;
    const h = Math.floor(next.minutes / 60);
    const m = next.minutes % 60;
    timeDiv.textContent = \`In \${h}h \${m}m\`;
  } else {
    scheduleDiv.textContent = "No more classes today!";
    timeDiv.textContent = "";
  }
}
update();
setInterval(update, 60000);

Step 2: Load the Extension

  1. Open your browser (Chrome, Edge, etc.)
  2. Go to the extensions page:
    • Chrome: chrome://extensions/
    • Edge: edge://extensions/
  3. Enable **Developer mode** (toggle at top right).
  4. Click **Load unpacked** and select the folder where you saved the files.
  5. The extension icon appears in the toolbar.
  6. Click it to see your schedule and countdown!

Notes:

  • You don't need an icon file since it's omitted here.
  • You can update your code files later and reload the extension in the extensions page.
